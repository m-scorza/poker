08

InkField

D15's instrument exception to D8
static-grain fallback shown · live version is WebGL
The showpiece — admitted as an instrument or not at all
  • What: a living ink-light field — slow fluid grain that leans toward the cursor, drifts with scroll, tints imperceptibly toward money/loss with the session's net.
  • Where: landing hero (primary), optionally Arena. Nowhere else, ever.
  • Never: violet, rainbow, plasma. It responds to the user's data or it doesn't exist.
  • Budget: raw WebGL, zero deps · DPR ≤ 1.5 · rAF pauses off-screen and tab-hidden · static-grain fallback for no-WebGL and reduced-motion.
09

The violet doctrine

D3 · the first law

Violet may touch

  • The halo — 22px ring, 44px over targets, mix-blend screen
  • Selection and focus rings
  • Active nav tick, tab underline, breadcrumb current
  • CTA arrows and the live pulse
  • The kicker's presence hairline
  • Matrix cell at the cursor only (scale 1.5 hover)

Violet must never touch

  • Surfaces, backgrounds, borders-at-rest
  • Charts or any data series
  • Matrix encoding (mixed is half-fill, not violet)
  • Severity (warning is faded loss, not violet)
  • Body text, headings, numbers
  • Anything decorative. If it isn't the user's presence or the way forward, it isn't violet.
