# scripts/

One-off utility scripts. Not part of the app build. Run from repo root.

| File | Purpose |
|---|---|
| `agent-kernel.cjs` | Local task-spool helper for validating protocol state, preflighting evidence, and completing scheduler tasks. |
| `parallel-runner.cjs` | Dry-run-first scheduler for non-overlapping agent worktrees. Use `--execute` only after reviewing the plan. |
| `agent-dispatch.ps1` | (Windows / PowerShell) Dispatches one spool task to a single worker CLI from `.agents/workers.json`: builds a kernel-aware prompt and dry-runs by default; `-Execute` launches the worker and logs to `.agents/runs/`. One task → one worker, vs `parallel-runner.cjs` which schedules many non-overlapping worktrees. `pwsh scripts/agent-dispatch.ps1 -List` |
| `regen-status.ts` | Regenerates / checks autogenerated blocks in `docs/product/STATUS.md`. `npm run docs:update` / `npm run docs:check` |
| `privacy-boundary-check.ts` | Static private/local boundary check for runtime network, telemetry, remote asset, native-share, and blocked SDK drift. `npm run privacy:check` |
| `install-hooks.js` | Installs the repo pre-commit hook during `npm install` / `npm ci`. |
| `surface-open-reports.ts` | Surfaces open/in-progress `docs/reports/*.md` at SessionStart. |
| `regen-reports-index.ts` | Regenerates the `docs/reports/README.md` index. |

All scripts assume the repo root as CWD.
